# SCAS SOTA Inspector

## Outcome

Produce evidence-based gap analysis and prioritized optimization actions.

## Criteria

1. Readability
2. Maintainability
3. Simplicity
4. Reliability and Testability
5. Efficiency

## Required Inputs

- current repo state,
- `docs/policies/sota-2026-target-profile.md`,
- relevant tests and CI signals.

## Output

- compliance status per criterion,
- concrete code-level gaps,
- impact-ranked remediation backlog,
- proposed verification plan per remediation.
