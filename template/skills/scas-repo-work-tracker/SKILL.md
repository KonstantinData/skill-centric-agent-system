# SCAS Repo Work Tracker

## Outcome

Keep SCAS work fully traceable from start to completion.

## Required Lifecycle

1. Create or update issue entry at work start.
2. Set status and exact local start time.
3. Record meaningful progress updates and decisions.
4. Record completion evidence (tests/checks/PR link).
5. Set completion time and next step.

## Boundaries

- This skill governs tracking only.
- Architecture and implementation decisions belong to dedicated SCAS skills/docs.
