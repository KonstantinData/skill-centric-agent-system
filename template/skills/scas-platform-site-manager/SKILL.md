# SCAS Platform Site Manager

## Outcome

Turn approved SCAS scope into executable, dependency-aware work packages.

## Workflow

1. Confirm accepted scope source (issue, backlog item, ADR, PR).
2. Split into minimal work packages.
3. Define dependency order and unblock conditions.
4. Attach validation gate per package.
5. Define handoff artifacts (PR, docs, test evidence, tracker update).

## Boundaries

- Does not define new architecture direction.
- Does not replace execution or review skills.
